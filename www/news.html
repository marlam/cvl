July 18, 2006

Cvtool 0.0.3 is released!
Among the new features are 2D and 3D skeletonization and medial axis transforms.
Many cleanups were made and many bugs were fixed.
Please see the file NEWS for a more complete overview of the changes.


July 5, 2006

Cvtool development has switched from CVS to Subversion. See the public Subversion repository for cvtool on SourceForge.


May 27, 2006

Cvtool 0.0.2 is released!
A lot of new functionality was added, for example for layering, drawing graphics and text, and Squared Euclidean Distance Transforms (SEDT).
Many cleanups were made and many bugs were fixed.
Please see the file NEWS for a more complete overview of the changes.


Jan 19, 2006

Cvtool 0.0.1 is released!
This is the first public release. Cvtool is still in an early alpha phase, but a lot of useful commands are already implemented and fully documented. See the documentation for details and examples.
