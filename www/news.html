July 26, 2008

New releases!

  • CVL and cvtool 0.2.5
    This release fixes some bugs and build problems, improves compatibility with various graphics drivers, and reduces graphics memory usage for some important operations. See the ChangeLog for details.
  • CVLView 0.3.1
    This release can generate videos from files with multiple data sets. It also improves usability, e.g. with a better handling of rotations in 3D view mode, and fixes some build problems. See the ChangeLog for details.
  • CVLTonemap 0.2.2
    This release uses much less graphics memory than older versions. Common HDR image formats such as RGBE, EXR, TIFF, PFM, or DCRAW can now be opened directly if the pfstools package is installed. Some bugs and build problems were fixed. See the ChangeLog for details.


May 17, 2008

New releases!
The license was updated to GPL version 3 or later for all packages.

  • CVL and cvtool 0.2.4
    This release mainly fixes some bugs and build problems. See the ChangeLog for details.
  • CVLView 0.3.0
    This release adds a 3D view mode for height maps, depth sensor data, and more. See the shiny new screenshot for an example. Additionally, some bugs were fixed. See the ChangeLog for details.
  • CVLTonemap 0.2.1
    This release fixes some build problems and minor bugs. See the ChangeLog for details.


January 16, 2008

New releases!

  • CVL and cvtool 0.2.3
    This release adds five new tone mapping methods for high dynamic range images, and new functionality to mix multiple data sources. Several bugs were fixed. See the ChangeLog for details.
  • CVLView 0.2.1
    This release fixes some bugs. See the ChangeLog for details.
  • CVLTonemap 0.2.0
    This release supports five additional tone mapping operators and fixes some bugs. See the ChangeLog for details.


October 18, 2007

CVLView 0.2.0 is released!
This release supports multiple data sets in a single file. The same viewing parameters are used for all data sets. Opening a different file resets the parameters. The range selection widget allows much finer control and analysis now. See the ChangeLog for details.


October 16, 2007

CVL and Cvtool 0.2.2 are released!
This release mainly fixes some bugs, but it also adds minor new features. See the ChangeLog for details.


October 3, 2007

CVLView 0.1.2 is released!
This release fixes some bugs. See the ChangeLog file for details.


September 9, 2007

Cvtool 0.2.1 is released, together with CVLView 0.1.1 and CVLTonemap 0.1.1!
These releases fix some bugs, including build failures, potential segmentation faults, and wrong computation results for some input images. See the ChangeLog files for details.
Additionally, CVLTonemap now handles postprocessing parameters for each tone mapping method separately, and allows to save and load parameter sets.


August 31, 2007

Cvtool 0.2.0 is released, together with CVLView 0.1.0 and CVLTonemap 0.1.0!
This is a complete rewrite. All computations are now done on the graphics processing unit (GPU). Additionally, all programs can now handle floating point data, e.g. for high dynamic range imaging.


November 6, 2006

Cvtool 0.1.0 is released!
Some bug fixes, performance improvements, and functionality enhancements were made.
Please see the file NEWS for a more complete overview of the changes.


July 18, 2006

Cvtool 0.0.3 is released!
Among the new features are 2D and 3D skeletonization and medial axis transforms.
Many cleanups were made and many bugs were fixed.
