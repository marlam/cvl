Cvtool is still in an early alpha phase. It certainly needs some testing, review, and possibly redesign. Nevertheless, many basic commands for stream and image manipulations are already available and fully documented.

Among the currently supported more advanced features are:

  • Three-dimensional, spatiotemporal smoothing of video streams
  • Support to create, manipulate and display stereoscopic 3D video streams
  • Advanced methods to estimate optical flow

The latest development version of cvtool is 0.0.1, released Jan 16, 2006.
There is not yet a stable version of cvtool.
