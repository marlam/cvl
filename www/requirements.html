Platforms

So far, cvtool has been tested on GNU/Linux, NetBSD, Solaris, and Windows.

Cvtool is written in C99 and uses POSIX features where C99 is not sufficient. Systems that lack C99 or POSIX features are supported through the use of gnulib modules and the GNU autotools.


Libraries

Cvtool needs the following libraries:

  • The GNU Scientific Library, version 1.6 or later.
  • The CAIRO 2D graphics library, version 1.0.0 or later.
