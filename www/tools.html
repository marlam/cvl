Volume handling

Some cvtool commands, like sedt or skeleton, can interpret a series of images as 3D volume data.

The pgmvol package allows to convert between a series of PGM images and volume data sets in the format used by Vol Tools and the Lib Vol library. See the IAPR Technical Committee on DISCRETE GEOMETRY (TC18) website for a collection of 3D data in the Vol format.

Pgmvol is available from the SourceForge project page and the Subversion repository of cvtool.
