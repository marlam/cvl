CVLTonemap is an interactive viewer and tone mapping tool for high dynamic range images.

It supports the pfs file format. The pfstools can be used to import other file formats.

Currently, only two tone mapping methods are implemented: the global Drago 03 method and the local Durand 02 method. Tone mapping is computationally expensive; you'll need a fast graphcis card to use this program interactively.

CVLTonemap requires the Qt toolkit, version 4.2 or newer.

Some example files for testing. These files were downloaded from the HDR image gallery at the MPI and converted to PFS format with the pfstools.
