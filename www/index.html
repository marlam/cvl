Cvtool is a computer vision tool.

It works as a filter: it reads streams of images from standard input and writes manipulated images to standard output. It understands both the NetPBM image formats (pbm, pgm, ppm, pam) and the YUV4MPEG2 raw video format used by the MJPEG Tools.

Cvtool integrates all its functionality into a single binary, and makes it available through commands. For example,
$ cvtool filter gauss --3d --k 3 < video.y4m > filtered-video.y4m
applies three-dimensional (spatiotemporal) Gauss smoothing to the video video.y4m and writes the result to filtered-video.y4m.

All the image manipulation and analysis functionality of cvtool is implemented in CVL, a computer vision library. CVL is currently included in cvtool. It is fully documented and can easily be used by other projects.
